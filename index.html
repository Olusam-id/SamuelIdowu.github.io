Covid-19 SQL-queries

This project demonstrates SQL proficiency through querying a COVID-19 database to extract and transform data for Tableau visualization. The SQL queries utilized aggregate functions (SUM, COUNT) with GROUP BY clauses, JOIN operations to combine infection data with population tables for percentage calculations. Additional queries employed window functions for time-series analysis, and potentially CTEs for complex multi-step calculations. The queries extracted data from multiple dimension tables (geography, dates, countries) and fact tables (infections, deaths, population) to create a denormalized dataset optimized for Tableau, showcasing essential ETL processes and data aggregation skills for business intelligence and analytics.

Covid-19 Dashboard

This Tableau dashboard presents a comprehensive COVID-19 analysis from 2020-2021, built from SQL-queried data, featuring four interconnected visualizations: a summary scorecard displaying key global metrics including total cases, deaths, and death rate percentage; a geographical heat map to show infection rates across countries worldwide; a horizontal bar chart comparing death tolls across continents with Europe showing the highest impact; and a multi-line temporal trend chart tracking infection progression over time for multiple countries with both actual and estimated values distinguished through color coding. The dashboard demonstrates advanced Tableau and effective data storytelling that transforms raw SQL-queried pandemic data into actionable visual insights.
